研究背景

由原简报的核心问题、选取理由和相关性整理，不替代原文背景综述。

P1 需要处理稀疏不规则观测与高频海量点的统一建模。Mamba/SSM 的线性时间特性很诱人，但如果不知道状态空间中哪些 mode 真正承载信号，剪枝、压缩和解释都容易变成事后猜测。 从本日报的选取理由看，新增 / arXiv 2026-07-13。它给出选择性状态空间模型中 per-mode 输出贡献和 drop-subset 误差的精确测量，能在高频诊断长序列上先判断哪些状态模式随输入迁移，再谈线性内存压缩。

核心问题

选择性 SSM 的状态维度是否可以按输入动态使用和剪枝？如果可以，怎样在不重新部署模型的情况下精确估计 drop-subset 输出误差？ 对当前主题的关键意义在于：最小实验是在现有 Mamba/SSM 时序 baseline 上记录各诊断通道、窗口和采样率下的 mode 使用，比较静态剪枝、按通道剪枝和按输入剪枝对 NLL、提前量、显存峰值和鲁棒性的影响。

方法与关键创新

论文利用对角状态矩阵，把每个通道输出精确分解为 per-mode contributions，并构造 per-layer、per-channel、per-window 的 Gram tensor，离线计算任意 mode 子集丢弃后的输出误差；随后做 input-scheduled mode pruning。 阅读时需要把方法设计与主要结果一起看：摘要报告在 Mamba-1 family 上相对误差达到 2.3e-7，能预测 4,464 个部署剪枝配置的误差；在 Mamba-1、Falcon-Mamba 7B 和 Mamba-2 上发现状态使用会随上下文迁移，半状态预算可匹配未剪枝模型。

实验结果

摘要报告在 Mamba-1 family 上相对误差达到 2.3e-7，能预测 4,464 个部署剪枝配置的误差；在 Mamba-1、Falcon-Mamba 7B 和 Mamba-2 上发现状态使用会随上下文迁移，半状态预算可匹配未剪枝模型。

局限性

作者明确说该 scheduler 需要第一遍读取窗口状态，当前只展示可实现的 headroom，不声称已带来实际部署计算或显存节省；聚变高频信号上的 mode 迁移还需验证。

总结

它适合先做 P1 的状态审计工具，再决定是否引入动态状态压缩。

研究相关性与启发

最小实验是在现有 Mamba/SSM 时序 baseline 上记录各诊断通道、窗口和采样率下的 mode 使用，比较静态剪枝、按通道剪枝和按输入剪枝对 NLL、提前量、显存峰值和鲁棒性的影响。

P3 的可见光图像与诊断时序通常不同步，且可能有多个视角或局部区域。简单逐帧编码无法利用“同一结构随时间变化”的证据，容易被单帧噪声误导。 从本日报的选取理由看，新增 / arXiv 2026-07-13 / MICCAI 2026。它把互补视角和纵向显式对齐结合，用于风险预测；适合作为 P3 可见光图像序列与诊断时序对齐融合的跨领域模板。

核心问题

如何同时利用多视角空间互补性和纵向时间变化，让模型关注风险相关的变化而不是静态外观差异？ 对当前主题的关键意义在于：最小实验是为可见光图像建立少量固定区域或磁面相关 ROI token，与诊断时序按 shot 时间对齐，比较无对齐 late fusion、显式纵向对齐和诊断 query 图像 ROI 三种方案。

实验结果

摘要报告其在 EMBED 与 CSAW-CC 数据集上优于既有乳腺癌风险预测方法，并在不同密度和癌症子组中保持改进；代码可用。

局限性
