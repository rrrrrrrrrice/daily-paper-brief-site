时序基础模型常在预测任务上做零样本评估，但在分类、补全、异常抽象或状态识别上仍需要任务微调。聚变诊断同样需要同一表征支撑破裂预警、缺失补全、状态分段和跨炮检索，单一预测目标不够。 从本日报的选取理由看，新增。它把 forecasting、imputation、classification 等多种时序任务放进同一免微调 TSFM，核心是多尺度 Transformer 与 Multi-Objective Temporal Masking；适合作为高低频统一建模的基础模型参照。

核心问题

如何在不为每个下游任务单独微调的情况下，让同一个时序模型同时保留点级细节、长程结构和多任务所需的归纳偏置？ 对当前主题的关键意义在于：可把现有高低频诊断窗口统一成多尺度 token，预训练外推、补全和全局状态识别三类目标，再比较 Zeus 式多目标预训练、单一 forecast 预训练和监督破裂模型在跨炮 AUC、提前量和缺失鲁棒性上的差异。

方法与关键创新

Zeus 使用 point-wise tokenization 与 U-shaped hierarchy 的多尺度 Transformer，在细粒度 token fidelity 与长序列效率之间折中；训练目标采用 Multi-Objective Temporal Masking，将外推、插值和全局抽象纳入统一预训练策略。 阅读时需要把方法设计与主要结果一起看：论文报告在五类代表性任务上，Zeus 在 tuning-free 设置下持续取得有竞争力结果；arXiv 摘要页注明论文已被 ICML 2026 接收。

实验结果

论文报告在五类代表性任务上，Zeus 在 tuning-free 设置下持续取得有竞争力结果；arXiv 摘要页注明论文已被 ICML 2026 接收。

局限性

摘要未显示其是否显式支持真实不规则时间戳、异步多采样率和高频海量诊断；Transformer 层级结构虽然降低负担，但对百万级点序列仍需结合压缩或流式机制。

总结

Zeus 值得作为 P1 多任务时序基础模型基线，重点看其 masking 目标能否迁移到聚变诊断。

研究背景

由原简报的核心问题、选取理由和相关性整理，不替代原文背景综述。

长序列时序预测往往依赖 self-attention 建模历史时间点依赖，但标准注意力对窗口长度呈二次复杂度。聚变高频诊断中，若每个采样点都进注意力层，显存会先于模型能力成为瓶颈。 从本日报的选取理由看，新增。SGA 明确针对标准 self-attention 随 look-back length 二次增长的时间和显存问题，用共享可学习矩阵加输入相关残差近似注意力分数；对百万长度诊断窗口很直接。

核心问题

能否利用时序注意力图中跨时间戳冗余和重复模式，在保留输入相关变化的同时避免 query-key 全量分数计算？ 对当前主题的关键意义在于：可在现有 Transformer 破裂预测或多通道 forecasting backbone 中只替换注意力为 SGA，固定 tokenizer 和训练策略，比较显存峰值、吞吐、AUC、提前量以及采样率变化下性能退化。

方法与关键创新

SGA 用共享可学习矩阵表示公共注意力模式，用输入相关残差表示样本特异变化，从而绕开标准注意力分数中的 query/key projection。论文称其在 look-back length 上实现线性时间和 score-matrix memory complexity。 阅读时需要把方法设计与主要结果一起看：作者在电力、金融、天气、医疗监测、人类活动和气候等九个公开真实数据集上测试，报告相较标准 self-attention 和轻量注意力变体有更好的推理效率，同时保持有竞争力的预测性能。

实验结果

作者在电力、金融、天气、医疗监测、人类活动和气候等九个公开真实数据集上测试，报告相较标准 self-attention 和轻量注意力变体有更好的推理效率，同时保持有竞争力的预测性能。

局限性

SGA 仍基于规则采样 benchmark，未证明可直接处理真实 Δt、通道异步或采样率突变；共享注意力矩阵可能对 regime 快速切换的诊断数据表达不足。

总结

这篇是 P1 长序列显存控制的工程候选，适合作为 Transformer 替换注意力层先做小实验。

异步事件序列常用条件强度函数描述，但训练和似然计算需要数值积分，复杂度和误差都不理想。聚变诊断中许多重要前兆不是规则点值，而是峰值、突变、饱和、报警、控制动作等事件。 从本日报的选取理由看，新增。它不直接建模条件强度，而建模累计条件强度函数，避免数值积分误差，并用插值/外推分离的 monotone alternating splines 提升复杂异步事件动态表达力。

核心问题

如何在保证累计条件强度单调性的同时，提高复杂时间动态的表达能力，并避免数值近似带来的不稳定？ 对当前主题的关键意义在于：可把 ELM 突变、辐射尖峰、控制动作、阈值报警和缺失触发编码为事件序列，用 MAS 估计未来危险事件强度，再与连续诊断 encoder 的风险分数融合，比较提前预警和误报率。

方法与关键创新

论文指出现有 Monotone Neural Networks 存在凸性限制、饱和限制和 CCIF 约束冲突三类结构瓶颈；MAS 将插值和外推组件分开，用 monotone alternating splines 建模 CCIF，理论上缩小 MNN 的不可约近似缺口。 阅读时需要把方法设计与主要结果一起看：摘要报告 MAS 在合成和真实数据集上优于现有方法，并兼顾拟合精度和外推泛化。

实验结果

摘要报告 MAS 在合成和真实数据集上优于现有方法，并兼顾拟合精度和外推泛化。

局限性

它解决的是事件到事件的 TPP 建模，不直接给出多通道连续高频信号融合方案；用于破裂预测时需要先定义事件抽取或把连续信号与事件过程联合建模。

总结

MAS 适合作为不规则事件通道的概率层，与连续时序 encoder 互补，而不是替代所有诊断建模。

核心问题

如何在统一模型中同时预测未来视觉、稀疏触觉动态和动作，并在真正接触或高价值局部动态出现时才增强副模态注意力？ 对当前主题的关键意义在于：对 TriPro-ReID，可用事件率、时间表面变化、RGB 模糊度和属性区域置信度构造 event-phase gate，让属性 query 只在事件可靠且与属性区域相关时增强事件 token，避免固定帧化和直接相加。

方法与关键创新

VT-WAM 用统一 flow matching 学习未来视觉预测、触觉形变预测和动作预测；提出 Asymmetric Mixture-of-Transformers attention 连接首帧视觉锚点与时间触觉动态，并用 contact-gated Action-Visual-Tactile Attention Guidance 让 action query 在接触阶段依赖触觉证据。

实验结果

论文在六个真实 contact-rich manipulation 任务上报告 71.67% 平均成功率，相比 Fast-WAM 提升 26.67%，相比 OmniVTLA 提升 35.84%；消融显示触觉形变动态建模和 contact-phase tactile attention guidance 都重要。

局限性

触觉不是原生 event camera，动作监督也不同于属性识别；RGB-Event 迁移时需要重新定义“contact gate”为事件密度、运动边界、曝光质量或属性相关区域 gate。

总结

VT-WAM 的价值在于动态副模态 gate 与非对称融合设计，可作为 P2/P3 的跨领域机制参考。

趋势判断

1. 长序列路线正在从“全量注意力”转向“多尺度、共享结构和事件概率层”

Zeus 用 U-shaped hierarchy 平衡点级细节和长程抽象，SGA 把注意力分数中的冗余模式共享化，MAS 则把稀疏事件交给连续时间概率模型。对 P1 来说，更合理的路线是高频连续信号走线性/层级 encoder，稀疏关键事件走 TPP，而不是把所有点硬塞进同一个 Transformer。

2. 多模态融合的重点继续从“加更多模态”转向“判断何时可信、何处相关”

VT-WAM 用 contact gate 控制何时依赖触觉，Bridge-WA 用变化图压缩视觉证据，CALM 在非配对数据中保留可解释跨模态关联。这些都支持一个判断：P2/P3 的关键不是把 RGB、Event、可见光和诊断时序直接拼接，而是先估计模态可靠性、局部相关性和缺配对齐条件。
