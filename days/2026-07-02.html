研究背景

由原简报的核心问题、选取理由和相关性整理，不替代原文背景综述。

复杂工程系统的时序预测很少是单变量问题。电网、聚变诊断和多通道传感网络都存在节点间耦合、时间自相关和运行风险约束；只给点预测容易在安全关键场景中过度自信。 从本日报的选取理由看，新增。它把时空图预测残差转成表格 foundation model 可校准的表示，并同时保留 sequential 与 relational 依赖；对 P1/P3 的诊断网络、跨通道风险区间和实时预警可信度很有迁移价值。

核心问题

能否在不为每个新任务重新训练校准器的前提下，为图结构多变量时序预测给出同时覆盖空间关系与时间序列依赖的可靠预测区间？ 对当前主题的关键意义在于：可迁移到聚变诊断时，把通道作为图节点、炮段或窗口残差作为校准样本，比较普通 conformal、STOIC 式关系-序列校准和 MC dropout 在 AUC、提前量、ECE、区间覆盖率、误报率上的表现。

方法与关键创新

STOIC 先用 STGNN 生成点预测，再把空间-时间残差转成 tabular representation，交给 tabular foundation model 进行 in-context 校准。这个设计把 conformal prediction 的覆盖保证、图预测的关系结构和 foundation model 的零样本校准能力拆开组合。 阅读时需要把方法设计与主要结果一起看：作者在合成仿真、电力需求和区域供热网络等 5 个 benchmark 上评估，报告 STOIC 在预测区间可靠性和鲁棒性上超过已有 conformal baselines。

实验结果

作者在合成仿真、电力需求和区域供热网络等 5 个 benchmark 上评估，报告 STOIC 在预测区间可靠性和鲁棒性上超过已有 conformal baselines。

局限性

论文应用是能源时序，不处理原生不规则时间戳、高频诊断压缩或破裂前兆的极端类别不平衡；如果基础点预测模型本身漏掉关键瞬态，后验区间校准无法恢复信号。

总结

这篇适合作为 P1/P3 的“风险区间层”候选，而不是替代主干预测模型。

研究背景

由原简报的核心问题、选取理由和相关性整理，不替代原文背景综述。

Grad-Shafranov 平衡重建是托卡马克控制、形状重建和稳定性评估的基础。传统求解器计算成本高，单装置机器学习 surrogate 又常常难以迁移到新几何或新装置。 从本日报的选取理由看，非当日新增 / 回看推荐。今天聚变直接新增不足，这篇仍是跨装置泛化最值得补读的聚变 AI 论文之一；它把“单装置 surrogate”推进到多几何预训练与少样本目标装置迁移。

核心问题

能否让神经算子在多种 tokamak-like geometry 上预训练后，仅用少量目标装置样本迁移到新几何，并保持物理一致的平衡场预测？ 对当前主题的关键意义在于：最小实验是用现有装置数据做跨炮段或跨装置迁移，把诊断时序、可见光图像 token 与平衡 surrogate 输出共同输入破裂预测模型，消融多几何预训练、少样本 fine-tune 和不确定性校准。

方法与关键创新

作者把平衡重建表述为 cross-device operator learning，用几何参数和 profile 参数直接映射到 poloidal flux field；基于 Solov'ev 解析族构造 8 种 tokamak-like geometry，并比较 FNO、WNO、MIONet、NOMAD、Sp2GNO 等神经算子和迁移策略。 阅读时需要把方法设计与主要结果一起看：历史已核验记录显示，多几何预训练显著优于单几何预训练；WNO 在 100 个目标标注样本下 mean relative L2 error 可低于 4%，完全 fine-tune 后低于 2%，并报告相对并行数值基线约 15 到 1124 倍推理加速。

实验结果

历史已核验记录显示，多几何预训练显著优于单几何预训练；WNO 在 100 个目标标注样本下 mean relative L2 error 可低于 4%，完全 fine-tune 后低于 2%，并报告相对并行数值基线约 15 到 1124 倍推理加速。

局限性

当前主要是 Solov'ev 解析平衡家族上的受控验证，不等价于真实装置诊断闭环；真实诊断噪声、边界不确定性、执行器耦合和破裂前兆仍未覆盖。

总结

这篇是 P3 跨装置泛化的聚变侧支点，适合和可见光-诊断融合模型共同设计 leave-one-device 评估。

研究背景

由原简报的核心问题、选取理由和相关性整理，不替代原文背景综述。

潜空间世界模型常用于高维观测下的规划，但测试时分布偏移会让冻结模型越滚越错。破裂预测也有类似问题：装置状态、诊断配置和等离子体运行区间随 campaign 变化。 从本日报的选取理由看，新增。AdaJEPA 在模型预测控制闭环中用下一状态转移做自监督测试时更新，只需少量梯度步即可重校准世界模型；对 P3 中跨装置/跨工况破裂预测的在线漂移修正很有启发。

核心问题

能否在闭环控制或在线预测过程中，用新观测到的状态转移自监督地更新潜空间模型，而不依赖额外专家示范？ 对当前主题的关键意义在于：最小实验是在破裂预测中只允许使用当前窗口之前的诊断和可见光观测，对冻结模型、campaign-level fine-tune、AdaJEPA 式小步自监督更新进行对比，报告 AUC、提前量、ECE、误报率和在线延迟。

方法与关键创新

AdaJEPA 在每次 MPC replanning 后执行第一段动作，利用观测到的 next-state transition 作为自监督适应信号，然后用更新后的模型重新规划；论文强调少量梯度步即可在闭环中持续重校准。 阅读时需要把方法设计与主要结果一起看：作者报告 AdaJEPA 在多种 goal-reaching 任务中显著提升规划成功率，并且每个 MPC replanning step 只需少至一步梯度更新。

实验结果

作者报告 AdaJEPA 在多种 goal-reaching 任务中显著提升规划成功率，并且每个 MPC replanning step 只需少至一步梯度更新。

局限性

原应用是机器人/控制 goal-reaching，不是异步多模态破裂预测；在线更新若无约束可能在异常前兆窗口中追随噪声或泄漏未来信息。

总结

这篇适合作为 P3 的在线自适应设计参考，重点读其闭环更新协议和防止冻结模型失效的思想。

多模态模型在开放式视觉问答、属性描述和语义识别中经常出现多个合理答案。RGB-Event 属性识别同样会遇到遮挡、低照度、事件稀疏、属性边界模糊和模态缺失，硬标签训练会把这些歧义压成单点判断。 从本日报的选取理由看，新增。CoMet 不靠重复采样生成答案，而用轻量后处理模块分解上下文歧义和多解性；对 P2 的属性不确定性、开放词表属性和模态质量门控是比硬标签更合适的监督信号。

核心问题

能否高效估计多模态输出的不确定性，并区分由任务上下文引起的歧义和由输入证据支持多个答案引起的多解性？ 对当前主题的关键意义在于：在 TriPro-ReID 上可把属性预测拆成外观证据不确定性、动态证据不确定性和属性多解性，比较普通 BCE、label smoothing、CoMet 式后处理头在 mA、F1、ECE、低照度和事件缺失场景下的表现。

方法与关键创新

CoMet 把不确定性分成 context-specific term 与 multiplicity-specific term，并训练轻量 post-hoc uncertainty module 估计两者；它避免 autoregressive answer generation 和 repeated sampling，因此推理开销比采样式不确定性低。 阅读时需要把方法设计与主要结果一起看：论文在开放式多模态 benchmark、幻觉检测和多选视觉问答上报告了优于既有不确定性估计基线的结果，同时保持较高效率。

实验结果

论文在开放式多模态 benchmark、幻觉检测和多选视觉问答上报告了优于既有不确定性估计基线的结果，同时保持较高效率。

局限性

原任务是 MLLM 输出校准，不是 RGB-Event 原生属性识别；如果属性标签空间固定且训练集本身没有歧义标注，需要额外构造软标签或属性文本候选。

总结

CoMet 适合迁移为属性级可靠性头和样本级融合权重，而不是直接替换 TriPro-ReID 主干。

研究背景

由原简报的核心问题、选取理由和相关性整理，不替代原文背景综述。

科学图像和诊断图像常具有明确物理成像机制。把它们当普通自然图像会浪费先验，也会在小样本、跨装置或跨传感器迁移时过拟合表面纹理。 从本日报的选取理由看，新增。它用线性复杂度 Mamba 编码器、物理散射先验 masked autoencoding 和跨域预训练处理大幅面遥感图像；对 P1 的长序列显存控制和 P3 的物理先验视觉 tokenizer 都有可迁移模块。

核心问题

能否用物理成像先验指导自监督遮挡任务，同时避免 Transformer 在大幅面图像上的二次复杂度？ 对当前主题的关键意义在于：最小实验是在可见光破裂图像或 ECEI/BES 图像上比较 ViT-MAE、Mamba tokenizer、物理区域 mask 的重建质量、下游 AUC、峰值显存和少样本跨炮泛化。

方法与关键创新

SAMBA 使用 Scattering-Guided Bidirectional Mamba 作为线性复杂度编码器，引入中间 class token 缓解长序列信息汇聚问题；SG-MAE 通过三层散射先验遮挡策略对齐 SAR 成像机制，并用 SpatialMix 增强跨区域交互。 阅读时需要把方法设计与主要结果一起看：作者报告 SAMBA 在 7 个下游 SAR 分类和检测数据集上多数指标达到 SOTA，且参数量少于 CNN 与 Transformer 基线；散射引导遮挡比标准 MAE 遮挡提升 few-shot transfer。

实验结果

作者报告 SAMBA 在 7 个下游 SAR 分类和检测数据集上多数指标达到 SOTA，且参数量少于 CNN 与 Transformer 基线；散射引导遮挡比标准 MAE 遮挡提升 few-shot transfer。

局限性

SAR 散射先验不能直接照搬到可见光等离子体图像或 Event 数据；需要重新定义符合诊断物理的遮挡单元，例如边界、辐射结构、磁面相关区域或局部光斑。

总结

这篇适合作为 P1/P3 的“线性视觉 tokenizer + 物理遮挡预训练”模板。

趋势判断

1. 新增论文更关注“可信度”和“在线适应”

STOIC 与 CoMet 都不是追求更大主干，而是把输出可信度拆成可校准的结构；AdaJEPA 则把部署后的自适应作为闭环过程的一部分。这对聚变诊断比单纯提高离线 AUC 更重要。

2. 物理先验遮挡和线性主干值得进入图像-时序融合实验

SAMBA 的启发是：科学图像预训练不应照搬自然图像随机遮挡。可见光等离子体图像、ECEI/BES 或红外诊断都应定义自己的物理区域 mask，并优先测显存和少样本迁移。
